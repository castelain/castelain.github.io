JS中检测数据类型的方式

  • 前端知识点
  • 2018-07-24

方式概述

  1. typeof：检测数据类型的操作符
  2. instanceof：检测某个实例是否属于某个类
  3. constructor：获取当前实例的构造器
  4. Object.prototype.toString.call：获取当前实例的所属类型的信息

方式详解

typeof

检查指定变量的数据类型，返回值是该变量的类型（字符串形式的）
语法：typeof 变量
typeof 操作符是检查一个变量是否是 undefined 、String、Number、Boolean这4中基本数据类型中的一种的绝佳工具。但是，如果变量是 null 或者 对象时， typeof 操作符就不能把他们区分出来，而都只会返回’object’，实际意义不大。

注意：

  1. typeof 无法区分出 数组和其他对象，因为返回值都是 ‘object’
  2. typeof 可以判断出函数’function’

instanceof

原理：检查右边指定的构造函数的原型对象是否在左边指定的对象的原型链上，返回结果为 true / false
语法：obj instanceof constructor
注意：
1.基本数据类型 instanceof constructor永远是 false
2.引用数据类型 instanceof Object永远是 true
3.可以区分出数组和其他对象
4.只能用于引用数据类型，不适用与基本数据类型
5.instanceof 会检查整个原型链。因此，对于同一个实例对象，可能会对多个构造函数返回 true

constructor

原理：obj.constructor —>(等价于) 对应原型对象.constructor属性
作用：得知某个实例对象，到底是哪一个构造函数产生的
语法：obj.constructor === constructor

Object.prototype.toString.call

作用：返回指定变量的数据类型，既适用于基本数据类型，也适用于引用数据类型
返回值：字符串”[object 类型]”
