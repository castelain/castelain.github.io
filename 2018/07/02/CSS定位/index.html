CSS 定位（Positing）

CSS 定位机制

CSS 有三种基本的定位机制：普通流、浮动和绝对定位.
除非专门指定，不然所有的框都是在普通流中定位的。

CSS position 属性

position 属性的取值有四个，这就影响到了元素框生成的方式。

  1. static
    元素框正常生成，位置排放就按照普通文档流的方式来进行。

  2. relative
    元素框相对于它之前所处的位置，偏移某个距离。元素仍保持其未定位前的形状，它原本所占的空间仍被保留。

  3. absolute
    元素框从文档流中被移除。相对于离它最近的包含块发生一定的偏离。它原本所占的空间将不被保留。

  4. fixed
    元素框从文档流中被移除。相对于视窗（一般就是指浏览器的窗口）进行定位
    。它原本所占的空间将不被保留。

说明:

  1. 相对定位可以看做为普通文档流定位模型的一部分，因为元素的位置相对于它在普通文档流中的位置。
  2. 绝对定位和固定定位，都脱离了正常的文档流，它们先前所占的空间将不会被保留。因为它们脱离了正常的文档流，所有它们的存在，对于其它正常流中的元素可看做是透明的。这就造成了：它们可能会遮盖住部分的正常文档流中的内容。
