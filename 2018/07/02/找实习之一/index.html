面经系列–找实习啊找实习（一）

    这是我的第一篇面经，当然也是我的第一次面试。现在是2018年4月13日，是上海汉得的二面的日子。今天的我，人生中的第一次面试就挂了。昨天的我，过了二面，超级开心，所以可能有些忘乎所以了（毕竟，一开始想着自己是一面都过不了的）。然后呢，今天的二面，却表现的非常非常非常的糟糕。有多糟糕呢？现在想想，自己都觉得糟糕得有些不可思议了。尽管现在的我很难过，但还是要痛定思痛不是，还有人等着我去包养呢！嘻嘻，加油准备下一次的面试吧！

ps: 上海汉得的一面是考察一些基本的知识点；二面是技术面，主要看项目。

心得体会

总结了以下二面很失败的原因，主要有以下几点：

  1. 准备工作不充分。明明二面就是看项目，看代码，却没有好好地整理一下自己的代码，导致代码看起来乱糟糟的，给面试官留下了很不好的影响。
  2. 自信心不足。明明学习了那么久的 js ，明明可以尽量聊聊的，却老说自己不大会，让面试官以为真的就不会（有时候，不能太耿直啊，要学会适当的吹一下）。
  3. 还是准备不充分，导致在现场演示的时候，自己都不是很熟悉操作，让面试官等得很不耐烦。
  4. 不会及时补救。当面试官问，你还有什么要展示的时候，居然傻傻的说没有了，完全没想要拿自己以前做过的东西来讲讲（真的是蠢啊）。
  5. 太紧张。这导致了语言逻辑很混乱，自己现在到觉得听不下去，可想而知，是有多差。

下次要注意的地方：

  1. 但被问到自己不大熟悉的东西时，不要急着回答不会，要尽量冷静地思考一下。实在不行就发散性思维想想。切记不要一张口就是不会，老说自己不行，面试官肯定也认为你不行了啊
  2. 为面试所做的准备一定要充分，要仔细研究招聘的要求，为每一个可能性都做足了准备。
  3. 最重要的一点，也就是：一定要有自信。要相信自己是可以的，要让面试官觉得，你是有能力在的。

ps: 不过，js 是真的该再好好的深入研究一下了啊。

面试题记录

  1. js 的数据类型有哪些？
    答： null、undefined、boolean、string、number.

  2. 不动一个 css 样式文件，要如何覆盖掉其中的一些样式？
    答：可以再写一个样式表，里面的放那些想要覆盖成的样式的，在link进去的时候，主要把它放在之前那个样式表后面。

  3. 如何将一个div元素隐藏掉？
    答： 当时想到的办法有：设置 display 属性为 hidden，或者用 js 的hide()方法。

  4. js 有哪些优点？
    答： 问到这个的时候，真的是蒙了。（这种开放性题目，不能说不知道啊，要好好想想，说点东西）
    下面为网上找的一些答案：
    在JavaScript这样的用户端脚本语言出现之前，传统的数据提交和验证工作均由用户端浏览器通过网络传输到服务器上进行。如果数据量很大，这对于网络和服务器的资源来说实在是一种无形的浪费。而使用JavaScript就可以在客户端进行数据验证。
    JavaScript可以方便地操纵各种浏览器的对象，可以使用JavaScript来控制浏览器的外观，状态甚至运行方式，可以根据用户的需要“定制”浏览器，从而使网页更加友好。
    JavaScript可以使多种任务仅在用户端就可以完成而不需要网络和服务器的参与，从而支持分布式的运算和处理。

5.对 vue.js 的理解。
答： Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

  1. 的作用？
    答：完全不会，当时乱答了一通。
    下来看的讲解：
    width: 可视区域的宽度，值可为数字或关键字 device-width。
    initial-scale: 页面首次显示时，可视窗口的缩放级别。1.0表示页面按实际尺寸显示，无任何的缩放。
    maxinum-scale=1.0,mininum-scale=1.0:也是表示缩放级别。前者取值为 1.0 ，则表示不可放大；后者取值为 1.0 ，则表示不可缩小。
    user-scalable: 是否可对界面进行缩放，取值为 no 则表示禁止缩放。
    参考网址：
    https://blog.csdn.net/u012402190/article/details/70172371
    meta 标签的作用就是更好地让我们的网页适配和响应不同分辨率的移动设备。
    1. js 中有哪些场景用到了回调函数？
      答： 天啊啊，太菜了，又一次不会！多学学吧
      一个回调函数指的是作为一个参数传递到另一个函数的函数，它将在父函数完成以后才被执行。
      异步请求的回调函数，点击事件的回调函数，同步调用的回调函数。
