hexo搭建博客（从零到一）

0. 日常叨叨

平常学习到的知识点多而杂，经常是当时理解，过不了几天就一片空白了。所以吧，古人说的“好记性不如烂笔头”确实中肯。可是吧，面对纷杂的笔记，不仅当时记录的时候很吃力，而且时候也不利于查阅。所以，在这良辰美景（国庆8天小长假）之际，我终于做了自己一直想做却没做的事（唉，人有时候就是这么自我矛盾）。

万事开头难，决定开始做就是成功的第一步！通过查阅网上各种资料、和小伙伴尬聊之后，最后选用了hexo来搭建自己的的一个博客。各种资料还算详细。但是，一路上，自己还算遇上了各种各样的坑，自觉应该记录下来（哈哈，以后还可以笑话笑话现在傻兮兮的自己）。好了，下面言归正传。

本篇仅为从零开始的基础篇，更多的高级设置暂不涉及，暂定内容为：

  1. 安装Git Bash
  2. 安装Nodejs
  3. 安装hexo
  4. 生成SSH以及添加到github
  5. 部署项目

  9. 后说
