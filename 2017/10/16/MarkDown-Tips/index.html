MarkDown –小贴士

MarkDown 的简要介绍

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。

表格中的所有内容，处理表格内容文字以外，都要确保使用的都是英文字符

如果表格相邻的上面一行有内容，一定要在这行内容与表格之间插入一行空行，这样，表格的格式才能生效（否则，会将表格内容作为普通文本输出的）
